To use this program simple select the type of tool you would like to paint/draw with (E.g. Roads) and draw on the canvas
When you are done drawing, place the starting and ending points for the Search algorithm you would like to test, and press run
If you would like to watch the algorithm scan in real-time, simply select the "Animated search" option!
